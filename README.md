# dostaffkin

Веб-приложение для заказа и отслеживания доставки.

## Функционал

- **Главная**: Обзор сервиса
- **Заказ доставки**: 
  - Расчёт стоимости по маршруту (карта, расстояние, сроки)
  - Выбор габаритов груза и срочности
  - Оформление заявки с контактами
- **Отслеживание**: 
  - Поиск отправления по ID
  - Просмотр статусов и маршрута

## Технологии

- **Фронтенд**: Angular 21, TypeScript, RxJS
- **UI**: Кастомная верстка (CSS)
- **Карты**: Yandex Maps API
- **Уведомления**: ngx-toastr
- **Сборка**: Angular CLI

## Установка и запуск

```bash
npm install
npm run start
npm run build
```

Приложение доступно по адресу `http://localhost:4200`.
